# アトランティック・テキサスホールデム

ポーカーバーで盛り上がる、テキサスホールデム×インディアンポーカーの飲みゲーム。
スマホ1台で回しながら遊べます。

## 遊び方

1. 人数（2〜6人）と名前を設定 → 「対局開始」
2. 各プレイヤーに伏せ札2枚＋場に共通カード7枚が配られる
3. **インディアンポーカー方式**：自分の伏せ札は見えず、他人の伏せ札と場のカードが見える
4. 順番にスマホを回しながら「勝負」or「フォールド」を選択
5. 全員決めたら清算

## ルール

### 勝者
- 勝負組のうち、**伏せ札2枚＋場の7枚から作る最強の5枚役**を持った人

### 飲む人
- **勝負して負けた人** → 自分の伏せ札の最大値を飲む（A=14, K=13, ..., 2=2）
- **フォールドしたが勝てた手だった人** → 同じく最大値を飲む（チキン罰）
- 勝者・賢いフォールダー → 飲まない

### 役の強さ（強→弱）
ロイヤルフラッシュ → ストレートフラッシュ → フォーカード → フルハウス → フラッシュ → ストレート → スリーカード → ツーペア → ワンペア → ハイカード

## 動かす

`index.html` をブラウザで開くだけ。スマホで遊ぶ場合はローカルサーバーで:

```bash
cd atlantic-texas-holdem
python -m http.server 8000
# 同じWi-Fi内のスマホから http://<PCのIP>:8000 にアクセス
```

外部依存なし、ビルドツール不要、Vanilla JS の単一HTMLファイル。

## デプロイ

[Cloudflare Pages](https://pages.cloudflare.com) や [Vercel](https://vercel.com) にリポジトリを繋ぐだけ。ビルドコマンド・出力ディレクトリ空欄でOK。

## 技術

- Vanilla JS, ES Modules不要
- HTML/CSSでカード描画（外部画像なし）
- 9枚から5枚を選ぶベスト役判定（126通り評価）
- Web Audio APIで効果音、`navigator.vibrate` で振動
