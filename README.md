# アトランティック・テキサスホールデム

ポーカーバー向け、複数端末でリアルタイムに繋がるテキサスホールデム×インディアンポーカーの飲みゲーム。

## 構成

- **Cloudflare Workers** + **Durable Objects** + **WebSocket**
- ルームコードで部屋を作って各自のスマホから接続
- 1台ずつスマホを額に当てて、自分の手札を他人に見せる方式

## 遊び方

1. 各自スマホでアクセス → 名前入力＋ルームコード（空なら新規発行）→ JOIN
2. 全員ロビーに揃ったらホストが START
3. **カウントダウン10秒**：スマホを額に当てる準備
4. **観察フェーズ45秒**：自分のスマホには「自分の手札2枚」と「共通カード7枚」が表示される。額に当てて他のプレイヤーに見せる。他人の手札を見て覚える
5. **判断フェーズ**：スマホを下ろし、共通カード7枚を見て「勝負」or「フォールド」を選択
6. **公開フェーズ**：全員の手札・最強5枚役・勝者・飲み杯数が表示される

## 役の強さ（強→弱）

ロイヤルフラッシュ → ストレートフラッシュ → フォーカード → フルハウス → フラッシュ → ストレート → スリーカード → ツーペア → ワンペア → ハイカード

## 飲み杯数

- **勝負した最弱役のプレイヤー**：勝負した人数 × 1杯
- **フォールドした人**：1杯
- **勝負した最強役のプレイヤー**（勝者）：0杯
- 全員フォールドした場合：全員1杯

## 開発

```bash
npm install
npm run dev   # ローカルで wrangler dev 起動
npm run deploy  # Cloudflare Workers にデプロイ
```

`wrangler.toml` で Durable Object と静的アセット（`public/`）の設定。`src/worker.js` がサーバ。

## デプロイ先

[Cloudflare Workers](https://workers.cloudflare.com)（Durable Objects 必要）

## ライセンス

個人で自由に使ってください。
